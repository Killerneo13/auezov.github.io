# Мұхтар Әуезов сайты

## 📖 Жоба туралы

Бұл веб-сайт қазақтың ұлы жазушысы, драматург, ғалым және академик 
Мұхтар Омарханұлы Әуезовтің өмірі мен шығармашылығына арналған.

Сайттың мақсаты — М.Әуезовтің әдеби мұрасын таныстыру, 
оның шығармаларын, өмірбаянын және қазақ әдебиетіне қосқан үлесін көрсету.

## 👤 Автор туралы

**Мұхтар Омарханұлы Әуезов (1897–1961)** — қазақ әдебиетінің классигі,
жазушы, драматург, ғалым, филология ғылымдарының докторы, профессор,
Қазақ КСР Ғылым академиясының академигі.

Оның ең танымал шығармаларының бірі:
- «Абай жолы» роман-эпопеясы
- «Көксерек»
- «Қараш-Қараш оқиғасы»
- «Қилы заман»
- «Еңлік-Кебек»
